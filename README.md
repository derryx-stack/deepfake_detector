# AI-Powered Deepfake Detection System

An AI-powered web application that detects manipulated images and videos using TensorFlow and Streamlit.

## Features

- Upload images and videos
- Detect whether media is Real or Fake
- Simple web interface built with Streamlit
- AI-powered prediction using TensorFlow

## Technologies Used

- Python
- TensorFlow
- Streamlit
- OpenCV
- NumPy
- Pillow

## Deployment

The application is deployed on Streamlit Cloud.

Live Demo:
https://deepfake-detector-derry.streamlit.app/

## Installation

Clone the repository:

```bash
git clone https://github.com/derryx-stack/deepfake_detector.git
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
